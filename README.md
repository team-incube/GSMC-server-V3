# GSMC-server-V3
GSM 인증제 관리 서비스 **GSMC** 입니다!
## 기여를 원한다면?
#### Commit Message Convention
| Type | Meaning |
| --- | --- |
| add | 새로운 코드나 파일을 추가하였을 때 |
| update | 기존의 코드를 수정했을 때 |
| fix | 버그를 수정했을 때 |
| delete | 삭제한 사항이 있을 때 |
| docs | 문서를 수정 |
| test | 테스트 관련 사항을 추가/수정하였을 때 |
| merge | 브랜치를 병합하였을 때 |
| init | 프로젝트 초기화 시 |

#### Branch Naming Convention
| Type | Meaning |
| --- | --- |
| main | 메인 브랜치 |
| develop | 개발 브랜치 |
| feat | 기능 추가 |
| hotfix | 긴급 수정 |
| refactor | 코드 리팩토링 |
